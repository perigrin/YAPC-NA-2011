Call for Trainers

YAPC has traditionally been a place that people could receive excellent training. The YAPC Organization team would like to officially open up our call for training. If you have experience as a professional trainer in a course that would be of interest to Perl programmers please contact us at talks@yapc2011.us

We are specifically looking for classes in any of the following areas:

  • Introduction to Perl
  • Intermediate Perl
  • Mastering Perl
  • Object Oriented Programming
  • Event Programming
  • Database Programming
  • Web Programming
  • Testing
  • Perl & Bioinformatics
  • Perl & Systems Administration / Programming
  • Care and Feeding of Camelids, Velociraptors and Lepidoptera

Space is limited, please contact us soon.
